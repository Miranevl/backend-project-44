<h1>Brian-games a series of 5 math mini-games</h1>
<hr></hr>

<h3>Requirements</h3>
<ul>
<li>Mac / Linux</li>
<li>Node.js version 18 or later</li>
</ul>
<hr></hr>

<h3>Install</h3>
<ol>
<li>$ git clone git@github.com:Miranevl/backend-project-44.git</li>
<li>$ cd backend-project-44</li>
<li>$ npm link</li>
</ol>
